# Kompote

My goal with this project is to create a todo/productivity app for my personal use utilising professional development techniques

I'll be using Kotlin and Jetpack Compose, which are entirely new to me

### Planned features
* Registering repeating and singular tasks
* Scheduling tasks and activities
* Send scheduled push notifications
* Shopping list
* JSON-based appdata

### Project standard
* Clean code
* Layer and package management
* Unit tests
* Advanced Git with branches
* Kotlin and Jetpack Compose
